# Contributing to localLOOP Docs Hub

Thanks for your interest in helping with the localLOOP documentation hub!

## Quick Start
- Create a feature branch.
- Open a pull request against `main`.
- Keep changes focused and well documented.

## Site Content
The site mirrors protocol artifacts from `loop-protocol`. Update the source repo when
possible and run the aggregation workflow to sync changes here.

## Contact
Questions? Email dev@mycel-ai.de or open a GitHub Discussion.
